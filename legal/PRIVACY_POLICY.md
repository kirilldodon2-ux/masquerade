## 1. What we collect
Masquerade collects *minimal* information:
- anonymized metrics (execution time, model usage),
- system logs (errors, load),
- no personal data, no cookies, no external trackers.

## 2. What we do NOT collect
We DO NOT store:
- prompts,
- uploaded images,
- output images,
- IP addresses,
- analytics IDs.

## 3. Third Parties
Model execution may run on:
- OpenAI,
- Anthropic,
- local inference backends (future),
but always without saving user content.

## 4. Security
All API keys and secrets are environment-protected and never exposed.
